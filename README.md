# Proyecto de Aseguramiento de Calidad (QA)

Bienvenido al proyecto de aseguramiento de calidad. Este repositorio contiene la documentación, casos de prueba, registros de ejecución y reportes de bugs para garantizar la calidad del software.

## Estructura del Proyecto

- **docs/** - Documentación de requisitos, análisis y glosario
- **test-design/** - Diseño de pruebas, planes y casos de prueba organizados por funcionalidad
- **test-execution/** - Registros de ejecución, corridas de prueba y evidencia
- **bugs/** - Reportes de errores encontrados durante las pruebas
- **ai/** - Prompts para IA y outputs generados
- **templates/** - Plantillas reutilizables para documentos

## Cómo comenzar

1. Revisa los requisitos en `docs/requirements/`
2. Consulta los casos de prueba en `test-design/test_cases/`
3. Registra las ejecuciones en `test-execution/`
4. Reporta bugs en `bugs/`
