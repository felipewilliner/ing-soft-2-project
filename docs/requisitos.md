# Requisitos

## Requisitos Funcionales
1. El sistema debe permitir a los usuarios iniciar sesión con un correo electrónico y una contraseña.
2. El sistema debe permitir a los usuarios restablecer su contraseña.

## Requisitos No Funcionales
1. El sistema debe ser accesible las 24 horas del día.
2. El sistema debe manejar al menos 1000 usuarios concurrentes.

## Requisitos de Seguridad
1. Las contraseñas deben ser almacenadas usando un algoritmo de hashing seguro.
2. El sistema debe utilizar HTTPS para todas las transacciones.
